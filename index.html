Project Source

Last Man Standing

An artificial agent that simulates three different bots that are trying to be the last bot alive. One bot is able to teleport randomly on the board; one bot randomly goes up, down, left, or right; and one bot that can go up, down, left, right, or can continue going in a straight line. If two bots land on the same square the second bot eats the first bot. The last bot standing wins. This project uses Python and the package pytest.
